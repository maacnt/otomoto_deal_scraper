# Otomoto Deal Scraper
Find the best value to price car based on set car filters to speed up finding the best deal for your dream car

# How it works
You go onto the website, go into advanced search, enter the filters you want to look your car for, run the main.py and paste your link into the terminal
code will then output a output.csv file with points based on the listing link and such

# Installation
Python3 or greater

1. Creating a python virtual environment
   ```bash
    python3 -m venv venv
    ```
2. Activate the virtual environment
   ```bash
    source venv/bin/activate  # On Linux/macOS
    .\\venv\\Scripts\\activate # On Windows
    ```
3. Install all dependencies from requirements.txt
    ```bash
    pip install -r requirements.txt
    ```
